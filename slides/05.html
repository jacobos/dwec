Document functions and properties(1)

  • createElement (document.createElement) → document.createElement (create a <html> element, an object)
    • Example, create a button: var btn = document.createElement("BUTTON");
  • createTextNode → document.createTextNode (create a text node, an object)
    • Example: var t = document.createTextNode("Hello World");
  • getElementById → document.getElementById (get an element by it's ID, an object)
    • Example: var a = document.getElementById("demo");
  • getElementsByClassName → document.getElementsByClassName (returns all the elements with a given class, an Array of objects)
    • Example: var s = document.getElementsByClassName("slide");

Document functions and properties(2)

  • getElementsByTagName → document.getElementsByTagName (returns all the elements with a given tag, an Array of objects)
  • document.body → Returns the body element of the document
  • querySelector → document.querySelector() → Returns the first element that matches a specified CSS selector(s) in the document
  • querySelectorAll → document.querySelectorAll() → Returns a static NodeList containing all elements that matches a specified CSS selector(s) in the document

Document functions and properties(3)

  • document.URL → Returns the full URL of the HTML document
  • document.images → Returns a collection of all <img> elements in the document
  • document.title → Sets or returns the title of the document
  • document.scripts → Returns a collection of <script> elements in the document
  • More properties and methods can be used on the HTML DOM Document Object

The DOM Element Object (1)

  • In the HTML DOM (Document Object Model), everything is a node:

    • The document itself is a document node
    • All HTML elements are element nodes
    • All HTML attributes are attribute nodes
    • Text inside HTML elements are text nodes
    • Comments are comment nodes

The DOM Element Object (2)

  • The HTML DOM Element Object, properties and methods
  • appendChild → document.getElementById("myList").appendChild(newListItem); (appends a Child to the element selected, in this case to the element with the Id "contenido"
    • Example: var newP = document.createElement("P"); document.getElementById("contenido").appendChild(newP);
  • childNodes → var chil = document.body.childNodes (returns an Array of all the nodes that append from another element, in this case the body)
  • children → var chil = document.body.children (returns an Array of all the elements (nodes that can have children), in this case of the body element)
  • cloneNode → var clon = document.getElementById("myList2").children[0].cloneNode(true) (clones a node into the given var)
    • Example: var node=document.getElementById("myList2")[0].cloneNode(true); document.getElementById("myList1").appendChild(node); // this makes a copy of node into the "myList1" element

    • The DOM Element Object (3)

  • firstChild → document.firstChild

    (returns the first child of an element, in this case of the <!DOCTYPE html> element

  • lastChild → document.lastChild

    (returns the last child of an element, in this case the <html> element)

  • id → var x = document.getElementById("myAnchor").id; (sets or gets the id of an element, in this case gets)
  • innerHTML → document.getElementById("demo").innerHTML = "Paragraph changed!";

    (gets or sets the HTML content of an alement)

    • The DOM Element Object (4)

  • parentNode → document.getElementById("item1").parentNode;

    (returns the parent node of the specified node)

  • removeChild → document.body.removeChild(document.body.childNodes[0]);

    (removes the given child of an element)

  • getAttribute → document.getElementsByTagName("a")[0].getAttribute("style");

    (returns the attribute specified of the given node)

  • setAttribute → document.getElementsById("contenido").setAttribute("class", "tipo_contenido");

    (sets a value to a given attribute of an element)
